# System Information

Commands to view system specs and usage.

## Examples

- `lscpu` – CPU info
- `lsblk -f` – Block device info
- `lshw -short` – Short hardware summary
- `free -h` – RAM usage
- `df -h` – Disk usage
- `du -sh folder/` – Folder size
- `uname -a` – System info
- `dmidecode -t memory` – Memory hardware info
