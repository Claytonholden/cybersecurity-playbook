# Working with Text

Use these commands to filter, sort, and manipulate text files.

## Commands

- `grep "error" logfile.txt`
- `sort data.txt`
- `uniq sorted.txt`
- `wc -l file.txt`
- `head -n 10 file.txt`
- `tail -f logfile.txt`
- `cut -d "," -f1 data.csv`
- `awk '{print $1}' file.txt`
- `sed 's/old/new/g' file.txt`
